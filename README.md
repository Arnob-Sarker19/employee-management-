# Real-time Chat App (WhatsApp Clone)

A full-stack chat application inspired by WhatsApp, built with:
- **Frontend**: React, Vite, Tailwind CSS, Lucide icons, Socket.io-client.
- **Backend**: Node.js, Express, Socket.io, Prisma ORM, SQLite.

## How to run locally

### 1. Backend
```bash
cd backend
npm install
npx prisma db push
npm run dev
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

The app will be available at http://localhost:5173.
